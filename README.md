# Networking-Project
Hands‑on networking labs built in Cisco Packet Tracer, covering subnetting, routing, ACLs and basic network security.
### Databridge Network Segmentation & ACL Lab

This project builds a small enterprise topology in Cisco Packet Tracer, using a core router (“Databridge”) to segment HR, Sales and Admin networks with separate subnets and to enforce access policies using extended ACLs. It walks through design, configuration and testing of connectivity before and after the ACL to show how layer‑3 filtering can protect sensitive admin resources.

**Tools used:** Cisco Packet Tracer, Cisco IOS CLI, Windows command line (ping/traceroute).

**Key topics covered:**
- Network topology design and IP addressing for multiple departments
- Router interface configuration and default gateway setup
- Extended ACL design to block HR → Admin traffic while allowing other flows
- Baseline and post‑ACL connectivity testing with ICMP
- Documentation of results and lessons learned
